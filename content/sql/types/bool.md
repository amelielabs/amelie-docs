## BOOL

Boolean type can be used in logical operations or as a column type.

* **`BOOL`**, **`BOOLEAN`**

	Accepted values are **`true`** and **`false`**.

---

```SQL
select true and false
[false]

select {"enabled": true}
[{
  "enabled": true
}]
```
